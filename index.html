Всплывающие окна

  • Способ 1.Один из самих лучших и простых. Использование display: table-cell + vertical-align: middle для вертикального центрирования.

    Преимущества:

    • Изменяемая высота
    • Верстка не едет при большом количестве текста в блоке
    • Кроссбраузерность

    Недостатки:

    • Сложная структура (дополнительная разметка)

  • Способ 2. Использование margin: auto. Для того чтобы это свойство работало по вертикали, необходимо задать position: absolute; top: 0; left: 0; bottom: 0; right: 0 и задать высоту.

    Преимущества:

    • Кроссбраузерность (включая IE 8-10)
    • Никакой дополнительной разметки, минимум стилей
    • Адаптивность
    • Работает для изображений

    Недостатки:

    • Должна быть задана высота
    • Рекомендуется задать overflow: auto, чтобы контент не расползался
    • Говорят не работает на Windows Phone

  • Способ 3. Центрирование блока с помощью jQuery. Весь функционал встроен в функцию $(window).resize(), которая выполняется при каждом изменении окна браузера и центрирует абсолютно спозиционированный блок изменяя значения top и left.

    Преимущества:

    • Кроссбраузерность
    • Простая разметка
    • Адаптивность

    Недостатки:

    • Более сложная реализация
    • Зачем подключать скрипт, если можно сделать через table-cell :)
  • Так же для реализации модальных окон можно использовать готовые решения таких фреймворков как bootstrap и foundation.
